В папке "Сайт" есть файл Index.html - это главная сайта.
В папке "JavaScript and JQerry" находяться скрипты к сайту.
В папке "Extra HTML" находяться дополнительные HTML странички.
В папке "CSS Style" находиться файл с CSS стилями для всего сайта.
В папке "Backgrounds and Images" находяться картинки и фоны используемые сайтом.
В папке "Addition" дополнительный материал для сайт (только в этой папке код не написанный мною).

2. 1024x600
3. 1024x768
4. 1024x800
5. 1024x900
6. 1280x720
7. 1280x768
8. 1280x800
9. 1280x1024
10. 1366x768
11. 1536x864
12. 1440x900
13. 1600x900
14. 1600x1024
15. 1600x1200
16. 1680x1050
17. 1920x1080
18. 1920x1200
19. 2560x900
20. 2560x1600

Сайт сделан в Retrowave и Synthwave стилях.
Также сайт сделан в розово-фиолетовых тонах.

Все HTML файлы были проверены на сайте http://validator.w3.org и ни одной ошибки не было выявлено.
